Certificate in Web Application Engineering

This certificate program prepares you for a web developer job which involves building modern web applications such as FaceBook, Twitter, Wikipedia etc. The curriculum begins with an introductory Computer Science course and then it prepares you to take a high level description of a web application, design it, develop the software and host a robust web application that can scale for thousands of users.


LEARNING OBJECTIVES - BY THE END OF THIS COURSE YOU WILL BE ABLE TO:

  • Develop code to solve computational problems

  • Analyse a high level description of a web application

  • Design a user interface for the web application

  • Implement web application using HTML, CSS, Java Script and Server Side code

  • Host the web application on the Internet


Course Work

  1. Object Oriented Programming - 4 credits

  2. Databases - 4 credits

  3. Web App Development (Front End) - 4 credits

  4. Web App Development (Server Side) - 4 credits

  5. Project - 8 credits
