DIPLOMA PROGRAM

Mobile App Development

Mastery in Building Android Apps



Franklin Jefferson University’s Mobile Applications Development diploma program consists of a series of Learn by Doing projects designed to help you attain a mastery in the job-relevant skills and build a portfolio to show prospective employers. The diploma program requires completing seven projects. You can see a list of these projects below.







Course Projects


BASIC LEVEL
PROJECT 1

Build a Single Screen App


Design and implement a simple app that displays information about a small business.

PROJECT 2

Court Counter App


Implement an app to track scores between two teams within a basketball game.

PROJECT 3

Report Card


Create a custom Java class to model the information in a student report card.


INTERMEDIATE LEVEL
PROJECT 4

Tour Guide App


Create an app to guide a user around a city or location of your choice!

PROJECT 5

News App


Create an app to display recent news stories on a topic of your choice using a web API.

PROJECT 6

Habit Tracker App


Create an app to track your habits! Supporting course content coming in August!


ADVANCED LEVEL
PROJECT 7

Inventory App


Create an app to track the inventory of a retail store, including current stock and supplier information.







Why Take This Diploma Program?


Our Diploma Program
