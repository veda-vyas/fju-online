Course Catalog



Digital Literacy (Basic)

Use the computer, browse the web, send emails, call on skype, use the search engine, and follow Internet safety guidelines.​


Digital Literacy (Advanced)

Create documents, Create spreadsheets, Create presentations, Share and collaborate online.


Intro Programming (Basic)

Read Java programs and interpret their behavior, Write Java programs to solve computational problems (variables, data types, statements, control flow, loops, functions, recursion and arrays).


Object Oriented Programming (Advanced)

Create complex data types using object oriented principles.​


Computer Systems (Basic)

Observe the number representations inside the computer, Interpret machine level code.​


Computer Systems (Advanced)

Observe memory referencing bugs, analyze performance, communication over networks.


Algorithms & Data Structures I

Iterable data types (Stacks, queues and bags)


Algorithms & Data Structures II

Sorting and Searching algorithms


Algorithms & Data Structures III

Graph processing (minimum spanning tree and shortest paths)


Algorithms & Data Structures IV

String processing (string sorts, tries, substring search, regular expressions & data compression)​


Databases

Design relational models, Create database tables, read, interpret and write SQL queries.​


Software Engineering

Explain the software development life cycle, use design processes and perform software testing.


Social Networks

Connecting people through technology, Online Collaboration and Crowdsourcing


Mobile Apps

Build a simple Android app.


Data Analytics

Data Manipulation, Data Analysis with Statistics and Machine Learning, Data Communication & Information Visualization.


Cloud Computing

Clouds, MapReduce, key-value stores, P2P systems.


Artificial Intelligence

Basics of modern AI and representative applications of AI.


Robotics

Probabilistic inference, planning and search, localization, tracking and control.


Computer Networks

Architecture & Principles, Switching, Routing, Naming, Addressing and Forwarding.


Security

Network security, Internet Worms, Spam, Denial of Service Attacks.
